# AI Daily Notes

Онлайн-ежедневник с AI-функциями для структурирования заметок.

## Цель
Создать рабочее веб-приложение для ведения заметок с AI-помощником.

## MVP функции
- Регистрация / вход (логин-пароль)
- CRUD заметок (по датам)
- Поиск по тексту
- Теги (ручные + AI)
- AI-функция (авто-тегирование или суммаризация)
- Экспорт в Markdown

## Технологический стек
- Frontend: Next.js
- Backend: Supabase (PostgreSQL + Auth)
- AI: OpenAI API
- Деплой: Vercel

## Структура проекта
- /artifacts - AI-артефакты (требования, дизайн, тесты)
- /docs - Документация
- /prompts - Шаблоны промптов
- .agents - Конфигурация агентов
