# Smart Requirements Tracker

Инструмент для бизнес-аналитиков по управлению требованиями с помощью AI.

## Цель
Помочь аналитикам декомпозировать требования с помощью AI-ассистента.

## MVP функции
- CRUD для Epics, Features, Stories
- AI-декомпозиция требований
- Экспорт в Markdown/PDF

## Технологический стек
- Frontend: Next.js (будет определено)
- Backend: Supabase (будет определено)
- AI: OpenAI/Claude API (будет определено)

## Структура проекта
/artifacts - AI-артефакты (требования, дизайн, тесты)
/docs - Документация
/prompts - Шаблоны промптов
.agents - Конфигурация агентов
